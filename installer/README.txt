SnipEasy Windows installer package

Install:
1. Extract this zip package.
2. Run Install-SnipEasy.cmd.
3. If Windows asks for permission to run the script, allow it.

Runtime:
SnipEasy currently requires Microsoft .NET 9 Desktop Runtime.
If the runtime is missing, the installer opens the official Microsoft download page.

Source code:
This package contains compiled application files and installer scripts only.
It does not include SnipEasy source files such as .cs, .xaml, .sln, or .csproj.
